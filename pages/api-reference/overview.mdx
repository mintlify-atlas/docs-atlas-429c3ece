## Components

| Component | Description |
|-----------|-------------|
| [`<Image>`](/pages/api-reference/components/image) | Optimized image component built on the HTML `<img>` element |
| [`<Link>`](/pages/api-reference/components/link) | Client-side navigation between routes |
| [`<Font>`](/pages/api-reference/components/font) | Optimize and self-host fonts |
| [`<Script>`](/pages/api-reference/components/script) | Load and optimize third-party scripts |

## File conventions

| File | Description |
|------|-------------|
| [`_document.js`](/pages/api-reference/file-conventions/document) | Customize the HTML document markup |
| [`_app.js`](/pages/api-reference/file-conventions/app) | Customize the App component used to initialize pages |

## Functions

| Function | Description |
|----------|-------------|
| [`getServerSideProps`](/pages/api-reference/functions/get-server-side-props) | Fetch data on each request for SSR |
| [`getStaticProps`](/pages/api-reference/functions/get-static-props) | Fetch data at build time for SSG |
| [`getStaticPaths`](/pages/api-reference/functions/get-static-paths) | Define dynamic paths for SSG |
| [`useRouter`](/pages/api-reference/functions/use-router) | Access the router object in components |
