---
title: NextRequest
description: API reference for NextRequest, which extends the Web Request API with convenience methods for cookies and URL access.
---

`NextRequest` extends the [Web `Request` API](https://developer.mozilla.org/docs/Web/API/Request) with additional convenience methods.

```ts
import type { NextRequest } from 'next/server'
```

## `cookies`

Read or mutate the `Set-Cookie` header of the request.

### `set(name, value)`

Set a cookie with the given value on the request.

```ts
// Given incoming request /home
// Set a cookie to hide the banner
request.cookies.set('show-banner', 'false')
// request will have `Set-Cookie: show-banner=false; path=/home` header
```

### `get(name)`

Return the value of the named cookie, or `undefined` if not found.

```ts
// Given incoming request /home
// { name: 'show-banner', value: 'false', Path: '/home' }
request.cookies.get('show-banner')
```

### `getAll()`

Return all values for the named cookie, or all cookies if no name is given.

```ts
request.cookies.getAll('experiments')
// [{ name: 'experiments', value: 'new-pricing-page' }, ...]

request.cookies.getAll() // all cookies
```

### `delete(name)`

Delete the named cookie from the request. Returns `true` if deleted.

```ts
request.cookies.delete('experiments')
```

### `has(name)`

Return `true` if the cookie exists on the request.

```ts
request.cookies.has('experiments')
```

### `clear()`

Remove all cookies from the request.

```ts
request.cookies.clear()
```

## `nextUrl`

Extends the native [`URL`](https://developer.mozilla.org/docs/Web/API/URL) API with Next.js-specific properties.

```ts
// Given a request to /home
request.nextUrl.pathname // '/home'

// Given a request to /home?name=lee
request.nextUrl.searchParams // { name: 'lee' }
```

### Available properties (App Router)

| Property | Type | Description |
|----------|------|-------------|
| `basePath` | `string` | The base path of the URL |
| `buildId` | `string \| undefined` | The Next.js build identifier |
| `pathname` | `string` | The URL pathname |
| `searchParams` | `Object` | The URL search parameters |

## Example: Middleware usage

```ts proxy.ts
import { NextRequest, NextResponse } from 'next/server'

export function proxy(request: NextRequest) {
  const url = request.nextUrl.clone()
  url.pathname = '/new-path'
  return NextResponse.redirect(url)
}
```
