---
title: useParams
description: API reference for the useParams hook, which reads dynamic route parameters filled in by the current URL.
---

`useParams` is a **Client Component** hook that reads the current route's dynamic params filled in by the current URL.

```tsx filename="app/example-client-component.tsx"
'use client'

import { useParams } from 'next/navigation'

export default function ExampleClientComponent() {
  const params = useParams<{ tag: string; item: string }>()

  // Route -> /shop/[tag]/[item]
  // URL -> /shop/shoes/nike-air-max-97
  // `params` -> { tag: 'shoes', item: 'nike-air-max-97' }
  console.log(params)

  return '...'
}
```

## Parameters

`useParams` does not take any parameters.

## Returns

`useParams` returns an object containing the current route's filled-in dynamic parameters.

- Each property name is the segment's name; each value is what the segment is filled in with.
- Values are either a `string` or array of `string`s depending on the type of dynamic segment (`[slug]`, `[...slug]`, or `[[...slug]]`).
- If the route has no dynamic parameters, returns an empty object.

| Route | URL | `useParams()` |
| --- | --- | --- |
| `app/shop/page.js` | `/shop` | `{}` |
| `app/shop/[slug]/page.js` | `/shop/1` | `{ slug: '1' }` |
| `app/shop/[tag]/[item]/page.js` | `/shop/1/2` | `{ tag: '1', item: '2' }` |
| `app/shop/[...slug]/page.js` | `/shop/1/2` | `{ slug: ['1', '2'] }` |
